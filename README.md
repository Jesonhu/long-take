# 一镜到底demo

## 主要依赖

+ AlloyTouch -- 滑动手势
+ Pixi.js -- 场景画布，精灵图等处理
+ gsap(TimelineMax、TweenMax) -- 动画时间轴控制

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
